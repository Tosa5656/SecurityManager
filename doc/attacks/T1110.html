T1110 - Brute Force

Attack Description

Brute Force attack involves systematically trying all possible combinations of passwords or keys to gain unauthorized access to a system.

How It Works

  • Automated credential enumeration
  • Use of common password dictionaries
  • Generation of combinations based on rules
  • Distributed attack from multiple IP addresses

MITRE ATT&CK ссылка

Защита с помощью Security Manager

smssh - SSH Attack Detection

  • Обнаружение: smssh monitor автоматически обнаруживает brute force атаки
  • Анализ логов: smssh parse-log /var/log/auth.log
  • Конфигурация: smssh apply /etc/ssh/sshd_config для отключения password auth

Рекомендации

  • Использовать ключевую аутентификацию вместо паролей
  • Внедрить fail2ban для блокировки IP
  • Установить лимиты на количество попыток входа
  • Использовать сложные пароли
