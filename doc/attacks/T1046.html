T1046 - Network Service Scanning

Описание атаки

Сканирование сети для обнаружения доступных сервисов, открытых портов и потенциальных уязвимостей. Атакующие используют различные инструменты сканирования для картирования сети и идентификации целей.

Принцип работы

  • Сканирование TCP/UDP портов
  • Определение версий сервисов (banner grabbing)
  • Обнаружение открытых портов и служб
  • Картирование сети и топологии
  • Идентификация потенциальных уязвимостей

T1046 - Network Service Scanning

Защита с помощью Security Manager

smnet - Network Monitoring

  • Сканирование портов: smnet scan - показывает открытые порты
  • Мониторинг соединений: smnet connection
  • Статистика сети: smnet stats
  • Регулярный мониторинг: помогает обнаруживать подозрительную активность

smssh - SSH Security

  • Мониторинг нестандартных портов: smssh monitor
  • Анализ логов: smssh parse-log /var/log/auth.log
  • Защита от сканирования: использование нестандартных портов SSH

smlog - Log Analysis

  • Мониторинг сетевой активности: smlog monitor
  • Анализ необычного трафика: smlog search "connection" /var/log/syslog

Рекомендации

  • Использование firewall для ограничения открытых портов
  • Регулярное сканирование собственной сети
  • Отключение ненужных сервисов
  • Использование IDS/IPS систем
  • Сегментация сети
  • Мониторинг сетевого трафика
